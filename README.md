# Trading Terminal

A modern, futuristic trading terminal with watchlist, order entry, positions/fills blotter, and real-time streaming updates.

## Tech Stack

- **React 18** + **TypeScript** + **Vite**
- **AG Grid** – dense grids with pinning, streaming updates, master/detail
- **Zustand** – state management
- **react-resizable-panels** – resizable layout

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
